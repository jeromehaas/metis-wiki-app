# Metis Wiki App

With metis, you can manage your notes and store them in your Mongo Atlas database.

<img src="public/readme/preview.png" width="700" style="margin: 20px 0" />


### How to run
```bash
git clone https://github.com/jeromehaas/metis-wiki-app
cd metis-wiki-app
nvm use
npm install 
npm run dev
```
